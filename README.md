# Annotiq AI React Site

This is the React.js version of the Annotiq AI website.

## Run locally

```bash
npm install
npm run dev
```

## Build for hosting

```bash
npm run build
```

The production files will be created in the `dist` folder.

## Free hosting idea

For a free public link without your GitHub username, upload this project to Netlify. Netlify will build it automatically and can give you a link like:

```text
https://annotiq-ai.netlify.app
```

The `netlify.toml` file is already included.

## Contact form

The contact/client forms use Netlify Forms. After the site is live on Netlify:

1. Open your site in Netlify.
2. Go to **Forms**.
3. Open **annotiq-contact**.
4. New client messages will appear there.
5. To receive email alerts, go to **Forms > Form notifications** and add your email address.
